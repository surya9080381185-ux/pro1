# Image to QR Code Converter

A full-stack application that converts uploaded images into sharable QR codes.

## Prerequisites

**Node.js is required to run this application.**
If you do not have Node.js installed:
1.  Go to [nodejs.org](https://nodejs.org/).
2.  Download and install the LTS version.
3.  Restart your terminal/command prompt.

## Installation

1.  Open your terminal in this project folder (`d:\project`).
2.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Application

1.  Start the server:
    ```bash
    npm start
    ```
2.  Open your browser and navigate to:
    `http://localhost:3000`

## Features

-   **Upload Image**: Upload PNG, JPG, GIF, or WEBP images.
-   **Generate QR**: Receive a QR code that points to the hosted image URL.
-   **Direct Link**: Scanning the QR code opens the image in your mobile browser.
-   **Download**: Download the generated QR code image.
